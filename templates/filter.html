The
Movie Recommender

How does this work?

Behind the works is a machine learning model utilizing NLP and word vecotorization that takes in movie titles from the movie dataset containing over 5000 movies, and runs the attributes of the movie through the model to find similar movies. The similarity is measured via the following information about the movie: directors, writers, cast, genre's, rating country, release year, duration, production companies, and description of the movie.
